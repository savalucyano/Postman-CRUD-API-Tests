# CRUD API Testing – Postman

## Project Overview

This project contains a Postman collection created to test CRUD (Create, Read, Update, Delete) operations through a REST API.

The collection validates the core functionality of the API by performing data retrieval, creation, modification, and deletion requests.

## Operations Tested

### Read

* Retrieve all records from the database
* Validate API response structure and returned data

### Create

* Add new records using POST requests
* Verify successful data creation

### Update

* Modify existing records
* Validate that changes are correctly applied

### Delete

* Remove records from the database
* Verify successful deletion of data

## Features

* CRUD operations testing
* HTTP request validation
* Response verification
* API endpoint testing
* Data management through REST API
